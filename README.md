# La Biblioteca Total · 博尔赫斯宇宙

> «El universo (que otros llaman la Biblioteca)…»
> 宇宙（别人称之为图书馆）。

**在线漫游 → <https://hannahlovegood.github.io/la-biblioteca-total/>**

一个可以走进去的博尔赫斯宇宙，单文件、离线逻辑、零依赖：

- **巴别图书馆**：无限生成的六角回廊。每间回廊的书架上混着三种书——真实引文（西中双语 + 出处，60+ 条）、无人写下的「未写之书」、只有乱码的「密码之书」；偶尔会撞见一件 ✦ 藏品。
- **万物索引**：博尔赫斯宇宙的 35 件藏品，分为物 · 人 · 地 · 念四类——阿莱夫、扎伊尔、沙之书、赫尼尔、富内斯、阿斯特里翁、特隆、悲王别墅、迷宫、镜子、老虎、遗忘……每件藏品一间专属回廊（考据条目 + 关联引文），藏品之间**双链互通**，反向链接自动补全。
- **目录查询**：查得到的词返回真实作品或藏品；查不到的词，图书馆会推测出一本「必然藏于某处、只是无人能找到」的书；查询它自己，会得到一本妄图收容图书馆的、图书馆里的书。
- **镜子**（整个世界左右翻转）与**阿莱夫**（页面底部一个几乎看不见的光点——找到它）。
- 配色取自博尔赫斯的失明：黄与金是他最后看见的颜色。

## 为什么不是「所有事物」的数据库

「包含宇宙内所有事物」恰恰是博尔赫斯反复书写的不可能之事——阿莱夫、富内斯的记忆、一比一的帝国地图，全是关于总目录必然失败的寓言。所以这个网站不做穷举，而是**表演无限**：有限的真实语料 + 无限的生成规则。它自己也是巴别图书馆里的一册。

## 本地运行

下载 `index.html`，浏览器直接打开即可（字体走 Google Fonts CDN，其余全部内联）。
